Para leer los datos del formulario, añade esta función a tu Apps Script y vuelve a implementar. Luego pega la URL aquí.

function doGet(e) { const ss = SpreadsheetApp.getActiveSpreadsheet(); const sheet = ss.getSheetByName("Respuestas"); if (!sheet) return out({ rows: [] }); const data = sheet.getDataRange().getValues(); const headers = data[0]; const rows = data.slice(1).map(row => { const obj = {}; headers.forEach((h, i) => obj[h] = row[i]); return obj; }); return out({ rows }); } function out(data) { return ContentService .createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); }
Es la misma URL que usaste en el formulario (después de añadir la función doGet y reimplementar).
